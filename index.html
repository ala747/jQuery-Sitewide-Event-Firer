Firing a callback after N time all across your website. Page 1!

The timer is running every second. One message should appear in the console every 6 seconds and another every 10 seconds, no matter you go to the other web page in this micro-website. Enjoy!

[Click to go to another page in the website.]

You can see the current example code below:

$(document).ready(function() {
    // fire fireMe() with all default options
    $.sitewideEvent( fireMe   );

    // fire an inline callback with custom options
    $.sitewideEvent(
        {
            cookieName: 'my_custom_cookie_name',
            fireAt: 10000, // 10s
            cookieExpires: new Date(new Date().setTime(new Date().getTime() + 24 * 60 * 60 * 1000)) // 24hs
        },
        function () {
            console.log("Hey! I'm the \"custom options\" example being fired every 10 seconds :)");
        }
    );
});
function fireMe () {
    console.log("Hi! I'm fireMe() being fired :)");
}
